# 🚀 프로젝트 기획서: EV 충전 인프라 SOS 대시보드 (지자체 정책 지원용)

본 프로젝트는 전국 자동차 등록 현황과 모빌리티 서비스 FAQ 데이터를 결합하여, 전국 17개 시·도의 전기차 충전 인프라 부족 현황을 시각화하고 지자체의 예산 집행 우선순위 결정을 돕는 **의사결정 지원 시스템**입니다.

---

## 1. 프로젝트 개요
* **타겟 사용자**: 지자체(시·도청) 기후환경과 및 친환경 정책 담당자
* **핵심 가치**: 데이터 기반의 '충전 불편 지수' 산출을 통한 정책 타당성 확보
* **개발 기간**: 10시간 (MVP 기준)
* **기술 스택**: Python, Pandas, Numpy, Bs4, Selenium, MySQL, Streamlit

---

## 2. 기능 요구사항 (Functional Requirements)

| 요구사항명 | 상세 설명 | 비고 |
| :--- | :--- | :--- |
| **데이터 파이프라인** | 공공데이터(CSV/API)를 수집하여 Pandas로 정제 후 MySQL에 적재 | Python 스크립트 자동화 |
| **충전 불편 지수 산출** | (시별 EV 등록수 × 가중치) + (FAQ 민원 수 × 가중치) 수식 연산 | Numpy 활용 |
| **SOS 핫스팟 히트맵** | 전국 17개 시·도별 지수를 색상 농도로 지도에 시각화 | `st.pydeck_chart` 활용 |
| **수요-공급 비교 차트** | 시 단위별 EV 등록수 대비 충전소/민원 현황 비교 랭킹 차트 | Altair/Plotly 활용 |
| **시민 불만 분석** | FAQ/리뷰 데이터에서 빈도 높은 키워드를 추출하여 시각화 | WordCloud 활용 |
| **가중치 시뮬레이터** | 사용자(주무관)가 민원/등록수 가중치를 직접 조절하여 결과 업데이트 | `st.slider` 연동 |
| **데이터 추출 기능** | 분석 결과 데이터를 CSV 파일로 로컬 PC에 다운로드 | `st.download_button` |
