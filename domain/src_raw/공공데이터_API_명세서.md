## 1. 전기차 충전기 상태 정보 (한국환경공단)

전국 전기차 충전소의 실시간 충전기 가용 상태를 조회합니다.

### 🔗 API 개요
| 항목 | 내용 |
| :--- | :--- |
| **기능** | 실시간 충전기 상태 및 운영 정보 제공 |
| **EndPoint** | `http://apis.data.go.kr/B552584/EvCharger/getChargerStatus` |
| **Method** | `GET` |
| **Response Type** | `XML`, `JSON` (기본값: XML) |

### 📥 요청 변수 (Request Parameters)
| 항목명 | 영문명 | 필수 | 타입 | 샘플 | 설명 |
| :--- | :--- | :---: | :---: | :--- | :--- |
| 서비스키 | `ServiceKey` | 필 | String | - | 공공데이터포털 인증키 |
| 페이지번호 | `pageNo` | 필 | Int | 1 | 조회 페이지 번호 |
| 결과수 | `numOfRows` | 필 | Int | 10 | 한 페이지 결과 수 (1~9999) |
| 상태갱신 | `period` | 옵 | Int | 5 | 조회 범위(분) (1~10) |
| 지역코드 | `zcode` | 옵 | String | 11 | 시도 코드 (서울: 11 등) |

### 📤 응답 항목 (Response Elements)
| 항목명 | 영문명 | 타입 | 샘플 | 설명 |
| :--- | :--- | :---: | :--- | :--- |
| 결과코드 | `resultCode` | String | 00 | 결과 상태 코드 (`00`: 정상) |
| 결과메시지 | `resultMsg` | String | OK | 상세 결과 메시지 |
| 기관ID | `busiId` | String | ME | 충전기 운영 기관 ID |
| 충전소ID | `statId` | String | 28260005 | 충전소 고유 코드 |
| 충전기ID | `chgerId` | String | 01 | 해당 충전소 내 충전기 순번 |
| 충전기상태 | `stat` | String | 2 | **상태 코드** (하단 코드표 참조) |
| 갱신일시 | `statUpdDt` | String | 2019082912 | 상태 정보 마지막 갱신 일시 |

### 📊 코드 정의표
#### [stat] 충전기 상태 코드
- `1`: 통신이상
- `2`: **충전가능** (분석 핵심 데이터)
- `3`: 충전중
- `4`: 운영중단
- `5`: 점검중
- `9`: 상태미확인
